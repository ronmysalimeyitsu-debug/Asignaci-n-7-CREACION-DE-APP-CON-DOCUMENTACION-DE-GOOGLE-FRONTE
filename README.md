# 🎓 Sistema de Gestión y Consulta de Documentación Legal

Bienvenido a la plataforma interactiva diseñada para la gestión, procesamiento y consulta eficiente de normativas y documentos legales de manera local y segura.

Entender y consultar marcos legales de forma estructurada es el salto definitivo entre la búsqueda manual dispersa y el control total sobre la información jurídica. Este proyecto documenta y simula un entorno robusto para la extracción y análisis de textos legales (como leyes de catastro y bienes públicos) en un espacio amigable, didáctico y libre de dependencias externas innecesarias.

## 🎯 Propósito del Proyecto

* **Educativo y Profesional:** Proporcionar herramientas automatizadas de análisis de texto, resúmenes y extracción precisa de artículos jurídicos.
* **Local y Seguro:** Procesamiento directo de archivos PDF en el sistema local, garantizando la privacidad y disponibilidad de la información.
* **Arquitectónico:** Servir como caso de estudio práctico (Asignación Académica) sobre cómo estructurar una aplicación web moderna aplicando buenas prácticas de desarrollo backend y frontend.

## 🚀 Tecnologías Utilizadas

* Python (Flask / PyPDF para procesamiento de documentos)
* JavaScript / HTML5 / CSS3
* Git y GitHub para control de versiones

## 🏗️ Arquitectura y Buenas Prácticas Aplicadas

Esta aplicación fue construida bajo estrictos estándares de ingeniería de software:
* **Modularidad Extrema:** Separación estricta de responsabilidades (*Separation of Concerns*), evitando componentes sobrecargados.
* **Procesamiento Robusto de Archivos:** Lectura estructurada de documentos PDF página por página mediante scripts especializados.
* **Gestión de Errores y Estados:** Manejo de excepciones ante archivos no encontrados o entradas inválidas.

## 🛠️ Instrucciones de Ejecución

1. Clonar el repositorio y navegar al directorio del proyecto:
   ```bash
   git clone https://github.com/ronmysalimeyitsu-debug/Asignaci-n-7-CREACION-DE-APP-CON-DOCUMENTACION-DE-GOOGLE-FRONTE.git
   cd Asignaci-n-7-CREACION-DE-APP-CON-DOCUMENTACION-DE-GOOGLE-FRONTE
   ```

2. Instalar las dependencias requeridas:
   ```bash
   pip install -r templates/requirements.txt
   ```
